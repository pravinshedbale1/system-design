## Rules

1. **Box 1 → 2**: Can retell the 2-minute story from memory + name key tradeoffs
2. **Box 2 → 3**: Can re-design the system from scratch (high-level, 10 min)
3. **Box 3 → 4**: Can handle deep-dive questions and curveballs
4. **Box 4 → 5**: Can design variations of the system (e.g., "now add feature X")
5. **Demotion**: If you can't narrate the story or forget key decisions → back to Box 1
